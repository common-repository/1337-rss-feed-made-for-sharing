== Installation ==

1. Upload `1337-RSS-Feed.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All settings must be filled out to work properly.

or

login to wp-admin
- Dashboard > Plugins > Add New
